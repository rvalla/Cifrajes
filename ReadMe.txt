Cifrajes

Cifrajes es el nombre que reciben un conjunto de herramientas para encriptar mensajes.
Son pequeños programas para jugar con distintas técnicas y alternativas de cifrado. No
sirven para uso profesional. Se programaron con el objetivo de ser usados en el aula
para estudiar criptografía en la escuela.
